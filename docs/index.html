How It Works

1

Fetch Riemann zeros

The non-trivial zeros of the Riemann zeta function, lying on the critical line Re(s) = 1/2.

2

Compute the explicit formula

Sum oscillatory terms from each zero to reconstruct the Chebyshev psi function — a prime counting signal.

3

Detect the spikes

Primes appear as jumps in the signal. Use statistical outlier detection to find them.
